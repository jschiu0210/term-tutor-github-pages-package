# 白話名詞小助教

這是一個純前端靜態網頁工具，可協助大學生快速理解專有名詞、複雜理論與企業 AI/工程術語。

## 檔案內容

- `index.html`：網頁主畫面
- `styles.css`：介面樣式
- `app.js`：詞庫、互動邏輯、圖像輔助說明

## GitHub Pages 部署方式

1. 建立一個 GitHub repository。
2. 上傳 `index.html`、`styles.css`、`app.js`。
3. 到 repository 的 `Settings` → `Pages`。
4. Source 選 `Deploy from a branch`。
5. Branch 選 `main`，資料夾選 `/root`。
6. 等待幾分鐘後，即可取得公開網址。

## 注意事項

- 本工具不需要後端。
- 本工具沒有使用 API key。
- 同學可直接用瀏覽器或手機開啟 GitHub Pages 網址使用。
